Persian Date Picker Element - Basic Usage

Simple examples of how to use the Persian Date Picker Element component with different configurations.

Basic Usage

Default configuration:
Selected date: None
<jalali-date-picker id="basic-picker"></jalali-date-picker>

Custom Placeholder

With a custom placeholder:
Selected date: None
<jalali-date-picker id="placeholder-picker" placeholder="تاریخ تولد خود را وارد کنید"></jalali-date-picker>

Custom Format

With a custom date format:
Selected date: None
<jalali-date-picker id="format-picker" format="YYYY-MM-DD"></jalali-date-picker>

Initial Value

With a programmatically set initial value:
Selected date: None
// Set initial value programmatically 
const initialPicker = document.getElementById('initial-picker');
initialPicker.setValue(1403, 3, 15);

Listen For Changes

With event listener for changes:
Jalali Date: None
Gregorian Date: None 
document.getElementById('event-picker').addEventListener('change', (event) => {
  const { jalali, gregorian } = event.detail;
  document.getElementById('jalali-result').textContent = jalali.join('/');
  document.getElementById('gregorian-result').textContent = gregorian.join('/');
});
