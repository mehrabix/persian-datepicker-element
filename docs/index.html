ویژگی‌های کلیدی

🗓️ تقویم شمسی (جلالی)

نمایش و انتخاب تاریخ بر اساس تقویم رسمی ایران، تبدیل خودکار به تاریخ میلادی

🎨 کاملاً قابل سفارشی‌سازی

استفاده از متغیرهای CSS برای شخصی‌سازی کامل رنگ‌ها، فونت‌ها، سایه‌ها و گوشه‌ها

🌗 پشتیبانی از حالت تاریک

تغییر خودکار به تم تاریک بر اساس تنظیمات کاربر یا تغییر دستی به حالت تاریک

📅 نمایش تعطیلات و رویدادها

نمایش تعطیلات رسمی ایران و افغانستان و امکان افزودن رویدادهای سفارشی

📱 واکنش‌گرا و موبایل-فرندلی

طراحی واکنش‌گرا برای نمایش بهینه در تمام اندازه‌های صفحه و تجربه کاربری عالی در موبایل

💪 پشتیبانی کامل از TypeScript

تعاریف کامل TypeScript برای API داخلی و خارجی، تکمیل خودکار و type safety
