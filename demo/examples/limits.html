محدودیت‌ها

کامپوننت Persian Date Picker دارای محدودیت‌های زیر است که باید در نظر گرفته شوند:

محدودیت‌های تاریخی

کامپوننت از سال ۱ تا ۱۵۰۰ هجری شمسی را پشتیبانی می‌کند.

// محدودیت سال min-year="1" max-year="1500"

محدودیت‌های انتخاب

می‌توانید محدودیت‌های انتخاب تاریخ را تعیین کنید.

// محدودیت انتخاب تاریخ const picker = document.querySelector('persian-datepicker-element'); picker.setMinDate(1400, 1, 1); // حداقل تاریخ: 1400/01/01 picker.setMaxDate(1400, 12, 29); // حداکثر تاریخ: 1400/12/29

محدودیت‌های نمایش

می‌توانید محدودیت‌های نمایش تقویم را تنظیم کنید.

// محدودیت نمایش show-week-numbers="false" show-holidays="true" show-today="true"

محدودیت‌های فرمت

فرمت‌های تاریخ باید مطابق با استاندارد تعیین شده باشند.

// فرمت‌های پشتیبانی شده format="YYYY/MM/DD" format="YYYY-MM-DD" format="DD/MM/YYYY"

نمونه استفاده

در این مثال، نحوه استفاده از محدودیت‌ها در تقویم نشان داده شده است.
