محدودیت حداقل و حداکثر تاریخ

می‌توانید با استفاده از ویژگی‌های min-date و max-date، محدودیت‌هایی برای انتخاب تاریخ تعیین کنید.

تاریخ‌های بین امروز و یک ماه بعد:
تاریخ انتخاب شده: انتخاب نشده
<persian-datepicker-element 
  min-date="[1402, 12, 10]" 
  max-date="[1403, 1, 10]"
></persian-datepicker-element>

تنظیم محدودیت‌ها به صورت پویا

در این مثال، می‌توانید به صورت پویا محدودیت‌ها را تغییر دهید:

تاریخ انتخاب شده: انتخاب نشده
// تنظیم محدودیت‌ها به صورت پویا با JavaScript
const picker = document.getElementById('dynamic-limits-picker');
picker.setMinDate(1402, 1, 1);
picker.setMaxDate(1403, 12, 29);

غیرفعال کردن روزهای خاص

می‌توانید تابع disabledDates را برای غیرفعال کردن روزهای خاص استفاده کنید:

غیرفعال کردن روزهای پنجشنبه و جمعه:
تاریخ انتخاب شده: انتخاب نشده
// غیرفعال کردن روزهای پنجشنبه و جمعه
const picker = document.getElementById('disabled-days-picker');
picker.setAttribute('disabled-dates', 'isWeekend');

// تعریف تابع isWeekend در JavaScript
function isWeekend(year, month, day) {
  const date = new Date(picker.jalaliToGregorian(year, month, day));
  const dayOfWeek = date.getDay();
  // 5 = پنجشنبه، 6 = جمعه
  return dayOfWeek === 5 || dayOfWeek === 6;
}

تعیین بازه تاریخی با تاریخ شروع و پایان

مثالی برای انتخاب بازه تاریخی با استفاده از دو تاریخ شروع و پایان:

تاریخ پایان:

// تعیین وابستگی بین تاریخ شروع و پایان
const startDatePicker = document.getElementById('start-date-picker');
const endDatePicker = document.getElementById('end-date-picker');

startDatePicker.addEventListener('change', (event) => {
  // تنظیم حداقل تاریخ برای تاریخ پایان به تاریخ شروع
  const { jalali } = event.detail;
  endDatePicker.setMinDate(jalali[0], jalali[1], jalali[2]);
});

endDatePicker.addEventListener('change', (event) => {
  // تنظیم حداکثر تاریخ برای تاریخ شروع به تاریخ پایان
  const { jalali } = event.detail;
  startDatePicker.setMaxDate(jalali[0], jalali[1], jalali[2]);
});
