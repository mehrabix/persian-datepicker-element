تغییر تم

Persian Date Picker از تغییر تم بین حالت روشن و تاریک پشتیبانی می‌کند.

تغییر تم خودکار

کامپوننت به طور خودکار با تنظیمات سیستم عامل شما هماهنگ می‌شود.

// تغییر تم خودکار persian-datepicker-element { --jdp-background: var(--background); --jdp-foreground: var(--foreground); --jdp-muted: var(--muted); --jdp-border: var(--border); }

تغییر تم دستی

شما می‌توانید تم را به صورت دستی تغییر دهید.

// تغییر تم دستی const picker = document.querySelector('persian-datepicker-element'); picker.setAttribute('darkmode', 'true'); // فعال کردن حالت تاریک picker.removeAttribute('darkmode'); // غیرفعال کردن حالت تاریک

ذخیره تنظیمات تم

شما می‌توانید تنظیمات تم را در حافظه مرورگر ذخیره کنید.

// ذخیره تنظیمات تم localStorage.setItem('theme', 'dark'); // ذخیره حالت تاریک localStorage.setItem('theme', 'light'); // ذخیره حالت روشن

نمونه استفاده

در این مثال، نحوه استفاده از تغییر تم در تقویم نشان داده شده است.
