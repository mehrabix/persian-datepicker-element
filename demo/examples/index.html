Persian Date Picker

Persian Date Picker یک کامپوننت وب برای انتخاب تاریخ در تقویم شمسی است. این کامپوننت با استفاده از Web Components ساخته شده و به راحتی در پروژه‌های وب قابل استفاده است.

ویژگی‌های اصلی

این کامپوننت از ویژگی‌های متعددی پشتیبانی می‌کند:

  • پشتیبانی کامل از تقویم شمسی
  • نمایش تعطیلات رسمی
  • پشتیبانی از حالت تاریک
  • قابلیت شخصی‌سازی کامل
  • دسترسی‌پذیری کامل
  • پشتیبانی از RTL

نصب و راه‌اندازی

برای استفاده از این کامپوننت، کافیست فایل JavaScript آن را در پروژه خود وارد کنید:

// نصب از طریق CDN
// نصب از طریق npm npm install persian-datepicker-element

استفاده پایه

ساده‌ترین روش استفاده از کامپوننت:

// HTML

رویدادها

کامپوننت از رویدادهای مختلف پشتیبانی می‌کند:

// JavaScript const picker = document.querySelector('persian-datepicker-element'); picker.addEventListener('change', (event) => { console.log('تاریخ انتخاب شده:', event.detail.jalali); }); picker.addEventListener('open', () => { console.log('تقویم باز شد'); }); picker.addEventListener('close', () => { console.log('تقویم بسته شد'); });

نمونه استفاده

در این مثال، نحوه استفاده از کامپوننت با تنظیمات پیش‌فرض نشان داده شده است.
